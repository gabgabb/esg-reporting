# Mini ESG Report

## Stacks
- Next.js
- Tailwind CSS
- Docker
- Zustand
- Chartjs

## Setup 
### Requirements
Docker and Docker Compose
### Build and Run
In the folder of the project

#### if you have makefile
```bash
   make dev ## Start and build dev container
```

#### if you don't have makefile
```bash
   docker compose up -d
```
