# Mini ESG Report

## Stacks
- Next.js
- Tailwind CSS
- Docker
- Zustand
- Chartjs

## Setup 
### Requirements
- Docker and Docker Compose
- yarn
### Build and Run
In the folder of the project

```bash
  yarn install
```

#### if you have makefile
```bash
  make dev ## Start and build dev container
```

#### if you don't have makefile
```bash
   docker compose up -d
```
